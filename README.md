# Python List Programs

This repository contains basic to intermediate Python programs based on **List operations**.

## Programs Included

- Alternate Prime Numbers
- Alternate Prime Without Count
- Reverse List using Negative Index
- Reverse using Position Index
- Reverse using Slicing
- Reverse using Two Pointer Method
- Sum Equals Target
- Sum Equals Target Pair

## Concepts Used

- Python Lists
- Looping (for loop)
- Conditional Statements
- Indexing
- Slicing
- Two Pointer Technique

## Purpose

These programs are written for practicing **Data Structures basics and Python logic building**.
